# Как открыть Shell (консоль) в Railway для выполнения команд

## Способ 1: Через активный деплой (рекомендуется)

1. **Откройте сервис up11**
   - В левом сайдбаре нажмите на "up11"

2. **Перейдите во вкладку "Deployments"**
   - Убедитесь, что вкладка "Deployments" активна

3. **Найдите активный деплой:**
   - Найдите зеленую карточку с надписью "ACTIVE"
   - Там должно быть написано "Deployment successful"

4. **Откройте Shell:**
   - Нажмите на кнопку **"View logs"** (зеленая кнопка справа)
   - ИЛИ нажмите на три точки (⋮) справа от деплоя
   - В выпадающем меню выберите **"Open Shell"** или **"Shell"**

5. **В открывшейся консоли выполните команды:**
   ```bash
   python manage.py migrate
   python manage.py collectstatic --noinput
   python manage.py createsuperuser
   ```

## Способ 2: Через вкладку Settings

1. **Откройте сервис up11**
2. **Перейдите во вкладку "Settings"**
3. **Найдите раздел "Shell" или "Console"**
4. **Нажмите "Open Shell"**

## Способ 3: Через кнопку в правом верхнем углу

1. **Откройте сервис up11**
2. **Посмотрите в правый верхний угол**
3. **Найдите иконку терминала/консоли** (обычно квадрат с символом `>_` или `$`)
4. **Нажмите на неё** - откроется Shell

## Способ 4: Через Railway CLI (если установлен)

Если у вас установлен Railway CLI:

1. **Установите Railway CLI** (если еще не установлен):
   ```bash
   npm i -g @railway/cli
   ```

2. **Войдите в Railway:**
   ```bash
   railway login
   ```

3. **Подключитесь к проекту:**
   ```bash
   railway link
   ```

4. **Выполните команды:**
   ```bash
   railway run python manage.py migrate
   railway run python manage.py collectstatic --noinput
   railway run python manage.py createsuperuser
   ```

## Визуальные подсказки:

- **"View logs"** - зеленая кнопка справа от активного деплоя
- **Три точки (⋮)** - вертикальное меню справа от деплоя
- **Иконка терминала** - обычно в правом верхнем углу
- **"Open Shell"** - опция в выпадающем меню

## Если Shell не открывается:

1. Убедитесь, что деплой успешно завершен (зеленая галочка)
2. Попробуйте обновить страницу (F5)
3. Попробуйте другой способ открытия Shell
4. Проверьте, что сервис запущен и работает

## Альтернатива: Автоматическое выполнение миграций

Если не получается открыть Shell, можно настроить автоматическое выполнение миграций:

1. **Откройте сервис up11** → **Settings**
2. **Найдите "Post Deploy Command"** или "Deploy Command"
3. **Добавьте команду:**
   ```bash
   python manage.py migrate && python manage.py collectstatic --noinput
   ```
4. **Сохраните** - миграции будут выполняться автоматически при каждом деплое
